Questionário

Questão 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

Resposta: O servidor web tem como função entregar arquivos estáticos (como imagens, CSS, HTML) para os clientes. Exemplos: Apache e NGINX
Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

Resposta: Ele é leve e eficiente, ideal para lidar com milhares de conexões simultâneas. Também pode atuar como proxy reverso e balanceador de carga.
Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

Resposta: HTTP não possui capacidade de criptografar dados, já HTTPS usa a criptografia SSL/TLS para garantir segurança. O HTTPS é recomentado por conta de sua proteção de dados e autentificação de sites.
Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

Resposta: Portas são canais de comunicação para diferentes serviços dentro de um dispositivo. A Porta 443 é utilizada para conexões HTTPS seguras, garantindo criptografia dos dados.
Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

Resposta: O endereço IP é o identificador único de um dispositivo em uma rede. Sua importância está atrelada à permissão de roteamento e comunicação entre máquinas.
Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

Resposta: É um servidor virtual privado que oferece mais controle, desempenho e segurança. Indicada quando o projeto exige mais desempenho e controle do que hospedagem compartilhada.
Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

Resposta: É um servidor que possui HTTPS automático, configuração simples com arquivos legíveis, desempenho moderno e foco em segurança.
Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

Resposta: Servidor ASP.NET Core. Indicado por ser uma plataforma moderna da Microsoft, com alto desempenho e suporte multiplataforma.
Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

Resposta: O cliente (navegador ou app) envia requisições ao servidor, que processa e responde com dados.
Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

Resposta: É um servidor de aplicação focado em Java. Ele executa Servlets e JSPs, sendo indicado para aplicações Java EE leves.
Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

Resposta: Servidor web: Entrega arquivos prontos (estáticos). Servidor de aplicação: Executa lógica de negócio, gera conteúdo dinâmico e se comunica com banco de dados
Questão 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.

Resposta: O domínio é um nome que facilita o acesso humano (ex: google.com) e que aponta para um IP. O DNS faz essa conversão para facilitar o acesso.
Questão 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?

Resposta: Vantagens como: Escalabilidade automática; alta disponibilidade; backup em tempo real e redução de custos com infraestrutura física.
Questão 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.

Resposta: Node.js - JavaScript e GlassFish – Java.
Questão 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?

Resposta: O HTTPS protege contra interceptações, roubo de dados e ataques do tipo manin-the-middle. O HTTP transmite dados sem criptografia, oferecendo riscos de segurança.
Questão 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.

Resposta: Um sistema de gerenciamento escolar, que precisa rodar aplicações personalizadas (como painéis internos, login de professores/alunos e relatórios), exigindo maior controle sobre o ambiente, segurança dos dados e disponibilidade constante. A hospedagem VPS permite instalar ferramentas específicas e ajustar configurações do servidor conforme as necessidades da escola.
Questão 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções

Resposta: IP serve para identificar o dispositivo. Porta serve para identificar o serviço específico rodando no dispositivo (como web, e-mail, FTP).
Questão 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

Resposta: Ele é fundamental para garantir a privacidade, segurança, autenticação do servidor e confiança do usuário.
Questão 19: Cite dois exemplos de servidores web e explique suas principais características.

Resposta: Apache: Servidor amplamente usado, flexível e compatível com PHP. LiteSpeed: Focado em performance, compatível com Apache, consome menos recursos e possui cache nativo, ideal para sites com alto tráfego.
Questão 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.

Resposta: São vários sites que compartilham o mesmo servidor. Indicado para projetos pequenos, blogs ou sites institucionais com pouco tráfego.
Questão 21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.

Resposta: Um servidor de aplicação executa lógicas de negócio e interage com bancos de dados. Já os servidores estáticos apenas entregam arquivos HTML, imagens etc.
Questão 22:Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.

Resposta: Gunicorn: Roda aplicações WSGI como Flask ou Django. Usa múltiplos processos (workers) para lidar com diversas requisições simultâneas.
Questão 23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?

Resposta: É um servidor corporativo da Red Hat para aplicações Java EE, com suporte a EJB, JMS, JPA e outros recursos empresariais.
Questão 24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?

Resposta: Porta padrão do FTP (File Transfer Protocol), usada para transferência de arquivos entre dispositivos.
Questão 25: Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

Resposta: Ele é compatível, pois aceita arquivos de configuração do Apache (como .htaccess), facilitando migração. Uma de suas vantagens é oferecer maior desempenho sem alterar estrutura existente.
Questão 26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?

Resposta: É um servidor web da Microsoft, com integração nativa ao Windows Server. Comumente adotado em ambientes corporativos e aplicações ASP.NET.
Questão 27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.

Resposta: Ele permite executar JavaScript no backend. Ideal para APIs, aplicações em tempo real como chats, sistemas com WebSocket.
Questão 28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.

Resposta: Um servidor de aplicação oficial para Java EE (Jakarta EE). Utilizado em sistemas empresariais Java com múltiplos módulos.
Questão 29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?

Resposta: São servidores de aplicação para Ruby on Rails. Puma: usa threads para lidar com várias conexões. Unicorn: multiprocessado, indicado para servidores com muitos núcleos.
Questão 30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?

Resposta: Essa escolha afeta diretamente o desempenho, escalabilidade e custo do projeto. Exemplo: Um sistema de agendamento médico online, que exige alta disponibilidade e segurança dos dados dos pacientes, se beneficia da hospedagem em nuvem, que oferece escalabilidade automática e maior tolerância a falhas, diferente de uma hospedagem local que exigiria infraestrutura própria e maior manutenção.
